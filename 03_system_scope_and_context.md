# Alcance y Contexto del Sistema

El sistema ERP interactúa con usuarios internos de la organización y con sistemas externos para el intercambio de información financiera.

El Administrador de Compras es el principal usuario del módulo de compras y es responsable de la gestión de productos y proveedores.  
El sistema también se integra con un sistema contable externo para el envío de datos de facturación y asientos contables.
