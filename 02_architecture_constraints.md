# Restricciones de Arquitectura

Las siguientes decisiones tecnológicas condicionan la arquitectura del sistema ERP:

- El backend del sistema se desarrollará en Java utilizando el framework Spring Boot.
- El frontend será una aplicación web tipo Single Page Application (SPA) desarrollada con React.
- La base de datos será relacional, utilizando PostgreSQL.
- La comunicación entre frontend y backend se realizará mediante servicios REST sobre el protocolo HTTPS.
- El sistema se implementará inicialmente como una aplicación monolítica.
